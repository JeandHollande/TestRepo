# Food Delivery App
This is a simple project to show how to implement the server sent event (SSE) connection through a food delivery application. 
In addition, we also use the observable design pattern.

# How to run?
Run the application (default port is 8080) and open in your browser. Use the 'Create order' button to create the events. 
The events will be added to the main page, and logging will take place in a separate css block.
