## Getting Started
Add a new directory containing a maven pom.xml, and select eclipse 'import maven project' menu item.<br>
Add a short description of the project to this file, using the Phoenix code editor.<br>
Mind: Don't use any other editor, I have tried that already, this seams to be the most convenient one.<p>
See for markdown syntax help: [MarkdownGuide](https://www.markdownguide.org/getting-started/)
